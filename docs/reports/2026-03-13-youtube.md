1.  **Metaスマートグラスのプライバシー問題**

2.  **AI利用が脳活動に与える影響**
    MITの研究が、AIアシスタント（大規模言語モデル、LLM）の利用が人間の認知能力に「認知負債」をもたらす可能性を指摘した。ChatGPTを利用したグループは、その後ツールなしでエッセイを書いた際、脳活動が著しく低下。AIが思考を代行することで、自力で考えるための脳の活動が減少し、批判的思考が低下するリスクがある。
    [Matt Wolfe - Is ChatGPT Making You Dumber?](https://www.youtube.com/watch?v=dobahObNLeA)

3.  **AI導入によるタスク拡大の真実**
    ハーバードビジネスレビューの研究によると、AIツールは仕事を減らすどころか、タスクを増やし、仕事のペースを加速させている。AIが知識不足を補うことで、従業員は専門外のタスクもこなすようになり、業務範囲が拡大。マルチタスクが増え、AIによる効率化で仕事のスピードに対する期待値も上昇する。
    [Matt Wolfe - Why ChatGPT Isn't Actually Making Your Life Easier](https://www.youtube.com/watch?v=retqU5rjzzI)

4.  **AIエージェントによるブラウザ自動化**
    AIエージェントが、長時間にわたるブラウザ自動化タスクに挑戦した。一時的なメールアドレス作成からTwitchへの登録、ストリームキー抽出、FFmpeg（動画のエンコード・デコード・変換などを行うソフトウェア）連携でYouTube動画をライブ配信。アンケートサイトでは質問と選択肢をDOM（ウェブページの構造）から抽出し、JavaScriptで一括回答することで1セントの収益を達成した。
    [All About AI - Long-Running AI Agent Browser Automation Tasks Is Here](https://www.youtube.com/watch?v=8RM-u7TkYpw)

5.  **AIエージェントのAWS自動攻略**
    AIエージェントがAWS（Amazon Web Services）コンソールを使ったブラウザ自動化の3つの課題に挑戦した。S3（Simple Storage Service）バケットで静的ウェブサイトを自動構築し、EC2（Elastic Compute Cloud）でLinux仮想マシンを起動、VNC（Virtual Network Computing）でグラフィカルリモートデスクトップにアクセス。複雑な仮想環境のセットアップやGUI操作の自動化に成功した。
    [All About AI - 3 AI Agent Browser Automation Challenges That Keep Getting Harder](https://www.youtube.com/watch?v=rodYMnGntdI)

6.  **AIによる自動取引戦略「オートリサーチ」**
    Andrej Karpathy氏の「autoresearch」プロジェクトに触発され、AIが自律的にビットコインの取引戦略を研究・改善するトレーディングボットが開発された。Polymarketという予測市場でビットコインの価格変動を利用したアービトラージ取引を自動で行い、人間が介入することなく最適な取引ロジックを見つけ出す。シミュレーションでは勝率100%を達成した。
    [All About AI - Karpathy's Autoresearch On My AI Polymarket Trading Bot](https://www.youtube.com/watch?v=kKucCudlHZs)

7.  **Anthropicの「Programmatic Tool Calling (PTC)」**
    Anthropicが、大規模言語モデル（LLM）エージェントのツール利用能力を大幅に向上させる「Programmatic Tool Calling (PTC)」を発表した。LLMが直接Pythonコードを生成し、そのコード内で複数のツール呼び出しや中間処理を完結させる。これにより、トークン消費を37〜50%削減し、処理速度が向上。Webページから必要な情報だけを抽出する「Dynamic Filtering」も導入された。
    [AI Jason - Anthropic killed Tool calling](https://www.youtube.com/watch?v=3wglqgskzjQ)
